# Generated output

This directory is the default destination for files produced by short,
user-facing workflows. Its generated contents are ignored by Git.

Running `Rscript scripts/quick_start.R` creates:

- `quick-start-reconstruction.html`; and
- `quick-start-reconstruction_files/`, containing its browser dependencies.

Keep the HTML file and dependency directory together when moving the
visualization. Both can be deleted and regenerated at any time.
