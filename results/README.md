# Results

This directory is for selected figures and tables that are intentionally
preserved because they support the README, a report, a poster, or a manuscript.

Use `figures/` and `tables/` for final outputs. Each result should have a
descriptive filename and documented provenance: the generating script,
parameters, source data, and relevant commit. Debug plots and bulk experiment
output belong in ignored working directories instead.

See the repository's [artifact policy](../docs/artifact-policy.md) for the full
selection and archival guidance.
