# Results

This directory is for selected figures and tables that are intentionally
preserved because they support the README, a report, a poster, or a manuscript.

## Selected figures

| Files | Purpose | Regeneration |
| --- | --- | --- |
| `figures/simulated-outbreak-input.{png,pdf}` | Synthetic observations shown on the poster | `Rscript scripts/demo.R` |
| `figures/example-reconstruction.{png,pdf}` | Posterior-mode reconstruction used by the README and poster | `Rscript scripts/demo.R` |
| `figures/testing-rate-vs-accuracy.{png,pdf}` | Poster comparison of testing rate and MAP accuracy | `Rscript scripts/build_poster_figures.R` |

PNG files are web previews; PDFs are the print-ready versions.

## Selected tables

`tables/testing-rate-r0-1-03.csv` and `tables/testing-rate-r0-2-5.csv`
contain the simulation replicates behind the testing-rate comparison. The
baseline table is generated by `scripts/run_accuracy_sweeps.R`. The R0 = 2.5
table is a retained snapshot from the extended poster sweep; its original RDS
object is preserved under `intermediate/sweeps/`.

The comparison figure is fully regenerated from these two committed CSV files.
The extended sweep should be consolidated into the main sweep script before an
archival software release.

Debug plots, alternate analyses, and bulk experiment output are retained locally
under ignored `intermediate/` directories rather than versioned here.

See the repository's [artifact policy](../docs/artifact-policy.md) for the full
selection and archival guidance.
